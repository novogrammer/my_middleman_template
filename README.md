my_middleman_template
=====================

# 自分用のMiddlemanひな形

ご自由にどうぞ。

＜＜gem周りのバージョンの依存関係が壊れています。余裕があるときに直します。＞＞

## 前提
* gem install bundler
* npm install -g bower

## 導入

`bundle install --path=vendor/bundle`

`npm install`

`bower install`

## サーバ

`bundle exec middleman`

## ビルド

`bundle exec middleman build`
